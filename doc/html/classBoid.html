Detailed Description

A class to contain all relevant data and methods for a boid in a flocking system. The class contains data such as position and velocity and methods that manipulate this data such as setTarget(), setSteering() and move(). the methods combine to form a flocking behaviour as set out by Craig Reynolds (1986).
